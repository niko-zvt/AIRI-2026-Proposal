# AIRI-2026-Proposal

Заявка на [Летнюю школу AIRI-2026](https://airi.net/ru/summer-school-2026/).

**Автор:** Животенко Николай

**Тема:** Гибридный подход к обратному дизайну металло-органических каркасов (MOF) через латентное пространство MOF/PMTransformer и proxy-свойства.

## Структура репозитория

- [`Research Proposal/`](./Research%20Proposal/) - текст заявки ([Zhivotenko-RP.md](./Research%20Proposal/Zhivotenko-RP.md) / [PDF](./Research%20Proposal/Zhivotenko-RP.pdf)), схема подхода и подборка статей.
- [`Experiment/`](./Experiment/) - код и результаты эксперимента: проверка согласованности латента PMTransformer со структурными proxy-свойствами MOF на 4500 структурах из CoREMOF, hMOF и QMOF.
- [`Utils/`](./Utils/) - утилита `md_to_pdf.py` для сборки PDF-версии заявки из Markdown по CSS-стилю.
